Game Requirements

  • Maximum number of players is 4.
  • Minimum number of players is 1.
  • Game Type:
    1. Maximum points decides winner (user input).
    2. Player with highest score at end of set rounds (user input).

Player Requirements

  • Throws per round is always 3.
  • Inner bullseye is 50 points.
  • Outer bullseye is 25 points.
  • Misses are allowed.
  • Keep track of all of the throw data per player.
